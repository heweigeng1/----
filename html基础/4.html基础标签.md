# 基础标签

初始化时生成的代码。

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
</body>
</html>
```

\<!DOCTYPE html> 声明为HTML5页面

\<head>\</head>  头部元素的容器， base,link,meta,script,style,title
